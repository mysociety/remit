Local development
-----
```
git clone https://github.com/mysociety/remit.git
cd remit
vagrant up
vagrant ssh
cd /vagrant
rails s -b 0.0.0.0
```

We use Vagrant to develop locally, it should install everything you need to
get a dev server up an running on http://localhost:3000
